コードだけでは意味が汲み取れない場合が多いため、コメントという形で解説します。

コメントは`/*`から始まり`*/`で終わります
```scala
/*
これは
helloという文字を出力するプログラム
*/
println("hello")
> hello
```

`/**/`を使用したコメントは複数行記述出来るのに対して
1行のみコメントすることが出来る`//`もあります。

```scala
// これはhelloという文字を出力するプログラム
println("hello")
> hello
```

コメントはプログラムの処理や実行時間に影響を与えないので気にせず記述して良いです。
